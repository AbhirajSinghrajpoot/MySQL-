## 🗺️ Course Curriculum & Roadmap

This repository is structured around the progressive difficulty of SQL concepts, moving from basic table creation to complex multi-table relationships.

### 🟢 Phase 1: The Basics (Currently Here)
* **Databases Intro:** Understanding Relational DBs, Tables, and Rows.
* **CRUD Operations 🔥:** The absolute core of database management.
  * `CREATE` (Databases & Tables)
  * `READ` (`SELECT`, `WHERE`, `ORDER BY`, `LIMIT`)
  * `UPDATE`
  * `DELETE`

### 🟡 Phase 2: Intermediate Querying
* **String Functions:** `CONCAT()`, `SUBSTRING()`, `REPLACE()`, `UPPER()`, `LOWER()`
* **Refining Selections:** `DISTINCT`, `LIKE`
* **Aggregate Functions 🔥:** `COUNT()`, `SUM()`, `AVG()`, `MIN()`, `MAX()`, `GROUP BY`
* **Data Types & Logical Operators:** `DATETIME`, `VARCHAR`, `DECIMAL`, `AND`, `OR`, `NOT`, `BETWEEN`, `IN`, `CASE`

### 🔴 Phase 3: Advanced Architecture & Relationships
* **Constraints:** `PRIMARY KEY`, `FOREIGN KEY`, `UNIQUE`, `NOT NULL`
* **Relationships 🔥🔥🔥:** Mastering One-to-Many logic.
* **JOINS 🔥🔥🔥:** The most critical section for backend engineering.
  * `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`

### 🟣 Phase 4: Capstone 
* **Instagram Clone Project 📸:** Designing a massive schema (`users`, `photos`, `comments`, `likes`, `follows`) utilizing complex joins and data analytics.
* **Node.js + MySQL:** Connecting the database to a live Express.js backend.
